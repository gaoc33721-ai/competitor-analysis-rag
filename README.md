# 智能营销素材问答与推荐系统 (Competitor Analysis RAG)

本系统演示了基于 RAG (检索增强生成) 的智能营销素材问答与推荐。
核心能力：自动提取竞品高优素材 → AI 标签化存储 → 自然语言语义检索。

## 版本说明

### v1.1.0 新特性 (当前版本)
- **强化防幻觉与品类对齐**：全面升级了 RAG 提示词工程，彻底解决“提问冰箱却推荐洗衣机”的张冠李戴问题，对跨品类推理实现零容忍。
- **品牌同义词自动识别**：支持中英文品牌名称等价识别（如“三星”自动匹配“Samsung”，“美的”匹配“Midea”），解决搜索遗漏。
- **精简直达的分析视角**：优化大模型回答风格，去除了机械罗列，直接输出推荐产品。且分析视角从“功能复述”升级为“视觉与文案亮点分析”，真正贴合营销业务人员需求。
- **UI 渲染容错机制**：彻底修复了前端在处理大模型不规范输出时可能导致的图片不展示、或错误展示不相关图片（历史记录污染）的 Bug。
- **定时任务与手动抓取双引擎**：后台管理系统新增“自动化定时抓取”配置（支持按天、按周、按月调度），并支持通过 ASIN 或关键词手动随时抓取补充新竞品。
- **品类动态解析与数据清洗**：流水线支持根据商品标题动态推断品类（不再全部归为 Electronics），并支持增量向量数据库同步，同时彻底清洗了旧版本的污染数据。

### v1.0.0 特性回顾
- 实现核心的 RAG (Retrieval-Augmented Generation) 架构。
- 接入 MiniMax 强大的 `abab6.5s-chat` 模型进行自然语言分析。
- 接入 Rainforest API 实现亚马逊美国站竞品素材的自动化抓取。
- 支持前端（Streamlit 聊天机器人）与后台（数据仪表盘）分离架构。
- 实现了基于 ChromaDB 的向量数据库存储与检索功能。

## 快速启动

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 配置环境变量：
确保根目录下存在 `.env` 文件，包含以下 API Key：
```env
MINIMAX_API_KEY=your_key
RAINFOREST_API_KEY=your_key
```

3. 启动前端用户界面 (Port 8501)：
```bash
streamlit run app.py
```

4. 启动后台管理界面 (Port 8502)：
```bash
streamlit run admin_app.py
```

5. 启动定时抓取流水线服务：
```bash
python scheduler.py
```
